# 抖音直播纯净模式助手 (Douyin Live Cleaner)

这是一个用于 Edge / Chrome 浏览器的插件，专门用于优化抖音网页版直播的观看体验。

## 🎯 功能特点

1. **自动收起侧边栏**：在进入全屏模式的前 3 秒内，自动点击收起右侧聊天栏，还你清爽画面。（手动展开后不再干扰）
2. **自动关闭弹窗**：智能识别并关闭直播间内的“点亮”、“送礼”、“集星”等诱导性弹窗。

## 📦 如何安装

由于插件未上架商店，需要手动安装：

1. 在本页面的右侧 "Releases" 点击下载最新的 `.zip` 压缩包。
2. 解压该压缩包到一个文件夹中。
3. 打开 Edge 浏览器，在地址栏输入 `edge://extensions` 并回车。
4. 打开左侧的 **"开发人员模式"** 开关。
5. 点击 **"加载解压缩的扩展"**。
6. 选择你刚才解压的文件夹即可。

## 🛠️ 开发说明

核心逻辑位于 `content.js`，基于 DOM 监听和文字识别技术实现，无需依赖具体的 CSS 类名，具有较强的抗更新能力。
